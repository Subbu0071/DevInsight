# DevInsight

DevInsight is a GitHub profile analyzer dashboard that helps users explore developer activity, repositories, coding patterns, and profile insights through a clean and responsive interface.

It takes a GitHub username as input, fetches public profile data using the GitHub API, and presents that data in an engaging dashboard with visual analytics and insight cards.

---

## Features

### Profile Overview
- Displays developer avatar, name, bio, location, and join year
- Gives a quick summary of the selected GitHub profile

### Core Metrics
- Public repositories
- Followers
- Following
- Developer Score

### Developer Insights
- Coding Rhythm
- Project Diversity
- Idea Velocity
- Project Survival Rate
- Tech Evolution
- Collaboration Score

### Visual Analytics
- Technology Distribution chart
- Developer Focus Radar
- Top Repository highlight
- Repository Explorer

### UI Highlights
- GitHub-inspired dark theme
- Animated hero logo with radar sweep effect
- Responsive layout for desktop, tablet, and mobile
- Tooltip-based explanations for insight cards
- Sticky header and polished dashboard cards

---

## How It Works

1. Enter a GitHub username
2. Click **Analyze** or press **Enter**
3. DevInsight fetches public data from the GitHub API
4. The dashboard updates automatically with profile insights and charts

## Insights Included

### Developer Score
A fun custom metric calculated using:
- number of public repositories
- followers
- total stars across repositories

**Formula:**

Developer Score = Repositories + (Followers × 2) + (Total Stars × 3)
---

## Coding Rhythm

Estimates whether a developer is more active during:

- Morning

- Afternoon

- Night

---

## Idea Velocity

- Shows how many repositories were created in the current year.

- Project Survival Rate

- Compares active repositories with inactive ones based on recent updates.

---

## Tech Evolution

- Compares older and newer repositories to understand how the developer’s technology stack has changed over time.

- Collaboration Score

- Checks forked vs original repositories to estimate collaboration patterns.
